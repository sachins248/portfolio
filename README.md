# Sachin Selvakumar — Portfolio

Personal portfolio website built with pure HTML and CSS. No frameworks, no dependencies, no build step.

**Live at:** https://portfolio-sachins248s-projects.vercel.app

---

## Stack

- **HTML/CSS/JS** — fully self-contained single file
- **Fonts** — Outfit (headings) + DM Mono (code/tags) via Google Fonts
- **Deployment** — Vercel

## Features

- Dark theme with smooth scroll animations
- Responsive layout with mobile hamburger menu
- Fade-in-up animations powered by IntersectionObserver
- Sections: Hero · Experience · Projects · Skills · Education · Contact
- Company logos for Verizon, Snapback Sports, and Kappa Theta Pi

## Deploying Changes

```bash
cd ~/portfolio
npx vercel --prod
```

Or connect this repo to Vercel for automatic deployments on every push.
