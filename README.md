# Sachin Selvakumar — Portfolio

Personal portfolio website built with pure HTML and CSS. No frameworks, no dependencies, no build step.

**Live at:** https://portfolio-sachins248s-projects.vercel.app

---

## Stack

- **HTML/CSS/JS** — fully self-contained single file
- **Fonts** — Outfit (headings) + DM Mono (code/tags) via Google Fonts
- **Deployment** — Vercel


## Deploying Changes

```bash
cd ~/portfolio
npx vercel --prod
```
